# MCP Todo Server
A .NET implementation of MCP Server.

# How to Run
* Run the application
```
dotnet run
```

# Configure your visual studio code
- Open Visual Studio Code
- Open opens the Command Palette (cmd+shift+P  or  F1)
- type:  MCP: Add Server
- Choose:  HTTP or Server-Sent Envent
- Enter the app URL
- Now you are ready!, Open VS Code Chat Window (agent mode) as Chat with it.

  Add task Play Football
